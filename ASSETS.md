# ASSETS.md — generated prop ledger

3D props are generated by the **Texas-drive TRELLIS pipeline** (RunPod network
volume `hrwxojdxoq`, US-TX-3, driven by `florida-man-simulator/tools/runpod_pod.py`
+ pod-side `batch_trellis.py`), NOT a fresh TRELLIS image. Reference product
images: `tools/gen-refs.ts` (xAI). Post-process to budget: `tools/postprocess-all.ts`
(Draco off → plain GLTFLoader). Committed GLBs live in `public/models/`; the
client loader (`src/render/loadProps.ts` `buildProp`) swaps them in for the
authored placeholders automatically. Collision geometry is hand-authored in
`src/game/physics/colliders.ts` and never generated.
